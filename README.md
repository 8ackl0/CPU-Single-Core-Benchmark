# CPU-Single-Core-Benchmark
CPU单核跑分天梯图
CPU Single Core Benchmark

本图收集了常见CPU型号单核跑分图，包括Intel 酷睿桌面处理器系列，Intel 酷睿移动处理器系列和AMD Ryzen处理器系列。
本图综合参考CineBench R15 和 GeekBench 的单核跑分数据，以i3-4100M的单核性能作为100%基准
